# 🚀 Dynamic Filter Panel Project

A modern and user-friendly web application that allows users to browse a product catalog and filter items in real-time. The application features a sleek "Glassmorphism" interface, providing instant feedback as users narrow down their search by category, price, rating, and availability.


## 🎯 What Does It Do?

The app allows users to navigate a product catalog with precision and ease:

Live Product Search:
Users can type into the search bar to find specific products instantly.

Multi-Criteria Filtering:
Filter products by specific categories (e.g., Electronics, Clothing) to find exactly what is needed.

Dynamic Price Control:
An interactive price slider allows users to set a maximum budget and see qualifying products immediately.

Rating-Based Discovery:
A dropdown menu enables users to filter items based on their minimum star rating.

Stock Availability Toggle:
Users can choose to view only the items currently in stock, with clear visual labels for "In Stock" and "Out of Stock."

Reset Functionality:
A dedicated button allows users to clear all filters and return to the full product list with one click.


## ⚙️ How Does It Work?

State-Driven Filtering Logic:
The project utilizes advanced array methods (like .filter()) to handle multiple simultaneous conditions, ensuring the UI stays in sync with user inputs.

Interactive UI Components:
Range sliders and checkboxes are linked to event listeners that trigger immediate re-renders of the product grid.

Modern CSS Architecture:
The layout uses Flexbox and Grid systems to ensure a responsive experience across all screen sizes, while utilizing blur effects for a "Glassmorphism" aesthetic.

Dynamic Card Generation:
Product data is mapped into reusable card components that display titles, prices, ratings, and stock status dynamically.
